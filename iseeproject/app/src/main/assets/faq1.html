SmartBucks FAQ

What is SmartBucks?

SmartBucks is a personal budget manager application that makes sure you are using the money wisely.

How to create user account?

Step 1:
Open the application SmartBucks and if you already have an account click Login or else click the “Sign up” button.
Step 2:
Fill the essential details such as the username, password, monthly income and the stable expenses in a month. These details will be added in the account details and can be updated at any point of time.
Step 3:
Click “confirm” button and you will redirected to the home page which means you have successfully created the account.

How to add expenses?

In the home page you can see the button “Add expenses”. Click the button and you will be redirected to the page where you can add the expenses.
Step 1:
Select the date from the date picker on which the expenses will be added.
Step 2:
Select the category of expense such as food, leisure , entertainment or you can create your own category under which you can add your expenses.
Step 3:
Select the mode of payment used for the particular expense.
Step 4:
Enter the amount that was spent in the provided text field.
Step 5:
Click confirm and then your expense will be reduced from the income. The graph will also be updated.

How to update my account details?

In the home page of the application when the menu option present on the top left of the application is clicked it provides various options.
Under those menu, there is an option called “Update Details”.
Once this option is clicked, you will be redirected to the update details page.
You can change the monthly income, user name and password along with the stable expenses.
In this update you can change only the things that you desire to update and leave the remaining fields unchanged or blank.
Once this is updated , click confirm and your details will be updated. This can be seen in the home page.

How to see different graphs?

In the homepage of the application you can visualise the expenses based on the timeline and category.
By default weekly graph is visible on the home page.
If you wish to see other visualisation, there are buttons such as Weekly, Monthly and category based graphs.
For knowing the expenses throughout the month, you can visualise by clicking the button “Monthly” in the homepage.
For knowing the expenses spent based on the category, you can click the “Category” button in the homepage.

How to generate monthly summary ?

Everyone wants to know their monthly expenses and SmartBucks provides an option to generate monthly expenses.
In the homepage there is a button “Show Expenses”. This will be used to view all the expenses that was added so far at any point of time.
In the menu option, there is an option named Generate report which will create the report of the expenses and then send the report to you by Email.
